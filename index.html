1

2

Модуль детекции

YOLOv8 нейросеть для обнаружения и классификации 10 типов дефектов

Применение бизнес-правил, определение сорта и веб-интерфейс оператора

Поток данных: Камера → Модуль 1 (захват изображения) → HTTP → Модуль 2 (детекция дефектов) → HTTP → Модуль 3 (сортировка) → Веб-интерфейс и база данных

Каждый модуль работает как независимый микросервис, что обеспечивает масштабируемость и отказоустойчивость системы.

Модули системы

Три специализированных компонента для полного цикла контроля качества

Автоматизированный захват изображений с промышленной линии

  • Оборудование: Промышленная камера HIKROBOT с разрешением 5472×3648 пикселей
  • Алгоритм: Обнаружение движения на основе OpenCV (фоновое вычитание, морфологические операции)
  • Управление: SDK MvCameraControl, автонастройка экспозиции и усиления
  • Интерфейс: Веб-панель управления с предпросмотром видеопотока
  • Интеграция: Автоматическая отправка изображений на сервер детекции по HTTP

Обнаружение и классификация дефектов нейросетью YOLOv8

  • Модель: YOLOv8m (medium) с дообучением на 10 000 размеченных изображений
  • Точность: mAP@0.5 = 80.2%, Precision = 79.5%, Recall = 72.5%
  • Классы дефектов: 10 типов (сучки, трещины, смола, гниль и др.)
  • Обработка: Асинхронная очередь (ImageQueue), поддержка CPU/GPU
  • API: REST эндпоинт для приема изображений и возврата JSON с результатами

Классификация сортов и веб-визуализация для оператора

  • Правила сортировки: Настраиваемые JSON-правила с учетом размеров и количества дефектов
  • Алгоритм: Определение сорта от лучшего к худшему на основе пороговых значений
  • База данных: SQLite для хранения истории с метаданными и изображениями
  • Веб-интерфейс: Реальный мониторинг, просмотр результатов, управление правилами
  • Экспорт: Полная история обработки с возможностью фильтрации и анализа
